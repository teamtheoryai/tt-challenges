# evidence/

Artifacts recovered while reconstructing the contractor's last days on the system. The box you're holding mirrors production, so these read across directly.

Read them the way an incident responder would: as leads, not conclusions. Some of what's in here matters; some is ordinary end-of-contract noise. Deciding which is which is part of the job.
