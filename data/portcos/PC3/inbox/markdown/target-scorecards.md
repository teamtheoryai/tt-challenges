# Target Scorecards by Role — Ridgeline Freight & Logistics

Role-level scorecards used to score candidates at hire / org DD and to define the **future-state bar** each seat must clear to deliver the value-creation plan. Scale 1–5. **Target** = thesis-ready score. **Min bar** = below this is a red flag. Weights sum to 100%.

> The gap between an executive's current score (leadership assessment) and the **Target** here drives **Future State Performance** and **Actionability**. For PC3 the dominant future-state factor is the **CEO-succession horizon**.

## How to read this against performance
- **Interview Performance** (baseline) was scored against these same weights at hire / org DD.
- **Future State Performance** = projected score vs. the **Target** column (gap to target). For the **CEO**, future-state intentionally **declines** over the horizon because the role transitions to the COO — this is a *plan*, not a performance miss.
- No PC3 executive is currently below a Min bar — consistent with the "healthy/on-plan" verdict. The only structural watch items are **CEO-succession timing** and **driver retention.**
